# Crakare Starter

## Setup Instructions
1. Install dependencies
```bash
npm install
```

2. Run dev server
```bash
npm run dev
```

3. Open http://localhost:3000 in your browser.

## Deploy to Vercel
- Connect this repo to your Vercel account
- Set env variables if needed
- Vercel auto-builds on every push
